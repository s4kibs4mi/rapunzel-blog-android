# Rapunzel Blog Android
Android client implementation of Rapunzel blog using Kotlin & gRPC.

#### Features
`[-]` Registration<br/>
`[-]` Initiate Verification via Email<br/>
`[-]` User Verification via Email<br/>
`[+]` Login<br/>
`[-]` Get User Profile<br/>
`[-]` Change User Status<br/>
`[-]` Change User Type<br/>
`[-]` Change Password<br/>
`[-]` Reset Password Request<br/>
`[-]` Reset Password<br/>
`[-]` Logout<br/>
`[-]` Create Post<br/>
`[-]` Update Post<br/>
`[-]` Delete Post<br/>
`[-]` List Posts<br/>
`[-]` Get Post<br/>
`[-]` Change Post Status<br/>
`[-]` Favourite Post<br/>
`[-]` Create Comment<br/>
`[-]` Update Comment<br/>
`[-]` Delete Comment<br/>
`[-]` List Comments<br/>
`[-]` Get Comment<br/>
`[-]` Favourite Comment<br/>
